# Routes

## Frontend Routes
* / 
  * Landing Page

* /login
  * Login Form

* /signup
  * Signup Form

* /sneakers/:sneakerId
  * Product Page

* /styles
  * Style Page

## Backend Routes
* GET
  * / Home Page

### API Endpoints
* users
  * POST /api/users 
  * Sign Up

* sneakers
  * GET /api/sneakers - Returns requested kicks
  * GET /api/sneakers/:id - Returns a specific shoe

* listings
  * GET /api/sneakers/:id/listings/:listing_id  Returns individual listing on a sneaker product page
